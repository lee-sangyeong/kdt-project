
# 현대 자동차 사전 프로젝트

## 프로젝트 소개
Python Tkinter를 활용하여 제작한 현대자동차 차량 정보 조회 GUI 프로그램입니다.

사용자는 연료 타입(가솔린, 디젤, 전기차)을 선택한 뒤 해당 차종 목록을 확인할 수 있으며,  
각 차량의 제원 정보와 이미지를 상세 화면에서 조회할 수 있도록 구현했습니다.

단순 텍스트 검색 프로그램이 아니라,  
차량 데이터를 **연료별 / 차종별 폴더 구조**로 관리하고,  
텍스트 제원 정보와 이미지 리소스를 함께 불러오는 방식으로 구성한 프로젝트입니다.

---

## 개발 목적
자동차 관련 정보를 단순히 텍스트로 정리하는 데서 끝나는 것이 아니라,  
사용자가 GUI 환경에서 직관적으로 차량 정보를 확인할 수 있도록 구현해보고자 했습니다.

또한 프로젝트를 통해 다음과 같은 역량을 익히는 것을 목표로 했습니다.

- Python Tkinter를 활용한 GUI 구성
- 차량 데이터를 폴더 구조로 체계적으로 관리하는 방법
- JSON / TXT 파일을 활용한 제원 정보 출력
- 이미지 파일을 연동한 상세 정보 화면 구현

---

## 주요 기능
- 연료 타입별 차량 분류
- 차종 목록 조회
- 차량 상세 제원 정보 출력
- 차량 이미지 표시
- 이전 / 다음 버튼을 통한 이미지 탐색
- 폴더 기반 데이터 관리

---

## 사용 기술
- Python
- Tkinter
- JSON
- os 모듈 (파일 및 폴더 경로 처리)
